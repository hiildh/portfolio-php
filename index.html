Desenvolvimento Web

Zombie Survivor Social Network (ZSSN)

Uma rede social onde sobreviventes de um apocalipse zumbi podem compartilhar recursos e informações. Utiliza Django e Django REST Framework para o backend e Vue.js no frontend.

  • Registro de Sobreviventes: cadastro de sobreviventes com informações detalhadas.
  • Dashboard do Sobrevivente: exibe detalhes do sobrevivente logado.
  • Relatórios: estatísticas de infecções e inventário.
  • Troca de Itens: permite que sobreviventes negociem recursos.

SNGPC - Sistema de Gerenciamento de Produtos Controlados

Uma plataforma para controle e rastreamento de medicamentos controlados, essencial para farmácias e distribuidoras. Esse sistema permite registrar todas as movimentações e manter a conformidade com as normas da Anvisa.

Nota: Este é um projeto privado, indisponível para acesso público.

Resumo Fiscal

Aplicação que processa arquivos de débitos fiscais gerados pelo ECAC e produz relatórios simplificados e fáceis de analisar.

Bot de Automação de Notas Fiscais com Selenium

Automação para baixar notas fiscais diretamente dos portais da Prefeitura de Teresina e SEFAZ usando o certificado digital da empresa. Ferramenta construída em Python e Selenium para otimizar o processamento e obtenção de documentos fiscais.
